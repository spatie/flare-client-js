# @flareapp/core

Environment-agnostic core for the Flare JavaScript SDK. Most users want
[`@flareapp/js`](../js) (browser) or [`@flareapp/node`](../node) (Node.js servers).
This package is the shared base that both consume; published so third-party
integrators can build against the same primitives.

Unstable until 1.0.0 — see the design doc in `docs/superpowers/specs/` for the
contract surface.
